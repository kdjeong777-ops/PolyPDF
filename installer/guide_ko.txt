PolyPDF 사용 안내
====================================================

[ 기본 사용 — API 키 불필요 ]
PDF 보기·편집·책갈피·검색·스크린샷·본문 읽기(음성)·화면 녹화·단어장 등
핵심 기능은 별도 키 없이 바로 사용할 수 있습니다.

[ 선택 기능: 온라인 사전·법령 검색 — 무료 API 키 입력 시 사용 ]
아래 기능은 무료 '오픈API' 인증키를 발급받아
   [ 도구 → 설정 → 인터넷 사전(단어장) ] 에 입력하면 켜집니다.
(영어 사전 Free Dictionary·예문 Tatoeba 는 키 없이 동작합니다.)

 1) 표준국어대사전 (국립국어원)
    - 발급: https://stdict.korean.go.kr  →  회원가입  →  '오픈API'  →  인증키 신청
    - 입력란: 설정 → '표준국어대사전 키'

 2) 우리말샘 (국립국어원 개방형 한국어 지식 대사전)
    - 발급: https://opendict.korean.go.kr  →  회원가입  →  '오픈API'  →  인증키 신청
    - 입력란: 설정 → '우리말샘 키'

 3) 온용어 — 전문용어 (국립국어원)
    - 발급: https://kli.korean.go.kr (언어정보나눔터) → '온용어' 오픈API 인증키
    - 입력란: 설정 → '온용어 키'

 4) 법제처 법령·고시 검색 (국가법령정보)
    - 발급: https://open.law.go.kr  →  OPEN API 신청  →  OC(이메일 ID) 발급
    - 입력란: 설정 → '법제처 OC'

[ PDF 기본 앱 설정 ]
설치 중 'PDF 기본 앱으로 설정'을 체크했다면, 설치 후 Windows '기본 앱' 설정창이 열립니다.
거기에서  .pdf  →  PolyPDF  를 선택하세요.
   ※ Windows 10/11 은 보안상 마지막 선택을 사용자가 직접 하도록 되어 있어,
     설치 프로그램이 자동으로 기본 앱을 바꾸지 않습니다(정상 동작).
   ※ 또는 PDF 파일 우클릭 → '연결 프로그램' → PolyPDF 를 선택해도 됩니다.

[ 업데이트 ]
도움말 → '업데이트 확인'. 새 버전이 있으면 백그라운드로 내려받아 종료 시 한 번에 교체됩니다.
( 설정의 '업데이트 자동 다운로드' 기본 켜짐 )

[ 화면 녹화 소리 ]
녹화에 소리가 필요하면  설정 → '녹화 테스트'  로 오디오 장치를 먼저 확인하세요.
